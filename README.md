# Morse Code Translator

A clean, modern styled UI web app for translating between text and Morse code.

 # Features

- **Bi-directional Translation** - Text to Morse or Morse to Text
- **Real-time Conversion** - Instant translation as you type
- **Audio Transmission** - Hear the Morse code played at 600Hz
- **Download Results** - Save your translations to a text file

## Tech Stack

- React + Vite
- Tailwind CSS
- Web Audio API

## How to run

npm install
npm run dev
Open the localhost link

## How to use

1. Type in either the left or right panel
2. Translation happens instantly
3. Click **TRANSMIT** to hear the audio
4. Click **DOWNLOAD** to save the result
5. Use **SWAP** to reverse input/output
6. Use **CLEAR** to reset everything
